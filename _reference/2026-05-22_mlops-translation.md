## Thematic grouping (suggested)

For quick navigation, the 35 lessons cluster into seven phases:

1. **Foundations (1–3):** What MLOps is, how it differs from DevOps, environment setup.
2. **Data & versioning (4–8):** Git for ML, data versioning, pipelines, feature stores, data validation.
3. **Experimentation (9–12):** Tracking, containerization, hyperparameter management, reproducibility.
4. **Training at scale (13–16):** Orchestration, distributed training, optimization, AutoML.
5. **Deployment (17–22):** Serialization, deployment patterns, APIs, serving platforms, A/B and canary rollouts.
6. **Production operations (23–27):** Monitoring, drift detection, alerting, logging.
7. **Automation & governance (28–35):** CI/CD, automated testing, continuous training, GitOps, governance, cost, security, platform architecture.
